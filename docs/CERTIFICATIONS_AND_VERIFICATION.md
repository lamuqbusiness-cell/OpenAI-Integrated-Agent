## üîí Part 1: Free SSL/TLS Certificates

### Option 1: Let's Encrypt (RECOMMENDED)
**Website:** https://letsencrypt.org
**Cost:** Free
**Renewal:** Automatic every 90 days
**Supported by:** Internet Security Research Group (ISRG)

#### Features:
- ‚úÖ Free, automated, and open
- ‚úÖ Valid for 90 days
- ‚úÖ Auto-renewal available
- ‚úÖ Wildcard certificates supported
- ‚úÖ Used by millions of websites

#### How to Get:
```bash
# Install Certbot
sudo apt-get install certbot

# Get certificate for your domain
sudo certbot certonly --standalone -d yourdomain.com -d www.yourdomain.com

# Auto-renewal setup
sudo systemctl enable certbot.timer
sudo systemctl start certbot.timer
```

#### Installation Links:
- Certbot Installation: https://certbot.eff.org
- Let's Encrypt Getting Started: https://letsencrypt.org/getting-started/
- ACME Protocol Guide: https://letsencrypt.org/docs/client-options/
